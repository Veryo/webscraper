Purpose of the application

Downloading product reviews from the Ceneo.pl website based on their individual codes

How it works

When valid code is executed, the product and its associated opinions are added to the database. This allows the user to view the opinions in a well-organized table, where they can be downloaded, filtered, and sorted. Additionally, visual representations of the opinions can be displayed in charts for further analysis

Libraries used

Django framework libraries like Django-filters | BeautifulSoup | Bootstrap | Chart.js

Approache

I utilized Beautiful Soup to scrape product reviews from Ceneo.pl, and then stored them in a database. In the views, I displayed the reviews, and for filtering, I incorporated the Django-filters library. For charts i used Chart.js
